The domain VectorSpaceBasis implements subspaces of vector spaces. A subspace is represented by a basis which is a list of vectors. Elements of basis are linearly independent so the empty list list describes the space with only the null vector. The scalar ring must be a field in order to avoid lattice (diofantine) questions.
