This package provides functions for the base case of the Risch algorithm.

extendedint: (Fraction UP, Fraction UP) -> Union(Record(ratpart: Fraction UP, coeff: Fraction UP), failed)
extendedint(f, g) returns fractions [h, c] such that c' = 0 and h' = f - cg, if (h, c) exist, “failed” otherwise.
infieldint: Fraction UP -> Union(Fraction UP, failed)
infieldint(f) returns g such that g' = f or “failed” if the integral of f is not a rational function.
integrate: Fraction UP -> IntegrationResult Fraction UP
integrate(f) returns g such that g' = f.
limitedint: (Fraction UP, List Fraction UP) -> Union(Record(mainpart: Fraction UP, limitedlogs: List Record(coeff: Fraction UP, logand: Fraction UP)), failed)
limitedint(f, [g1, ..., gn]) returns fractions [h, [[ci, gi]]] such that the gi's are among [g1, ..., gn], ci' = 0, and (h+sum(ci log(gi)))' = f, if possible, “failed” otherwise.
