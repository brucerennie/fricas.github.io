TranscendentalIntegration(F, UP)Â¶

intrf.spad line 301

  • F: Field
  • UP: UnivariatePolynomialCategory F

This package provides functions for the transcendental case of the Risch algorithm.

expextendedint: (Fraction UP, UP -> UP, (Integer, F) -> Record(ans: F, right: F, sol?: Boolean), Fraction UP) -> Union(Record(answer: Fraction UP, a0: F), Record(ratpart: Fraction UP, coeff: Fraction UP), failed)
expextendedint(f, ', foo, g) returns either [v, c] such that f = v' + c g and c' = 0, or [v, a] such that f = g' + a, and a = 0 or a has no integral in F. Returns “failed” if neither case can hold. Argument foo is a Risch differential equation function on F.
expintegrate: (Fraction UP, UP -> UP, Fraction UP -> Record(answer: Fraction UP, logpart: Fraction UP, ir: IntegrationResult Fraction UP), (Integer, F) -> Record(ans: F, right: F, primpart: F, sol?: Boolean)) -> Record(answer: IntegrationResult Fraction UP, a0: F)
expintegrate(f, ', foo) returns [g, a] such that f = g' + a, and a = 0 or a has no integral in F; Argument foo is a Risch differential equation solver on F.
expintfldpoly: (LaurentPolynomial(F, UP), (Integer, F) -> Record(ans: F, right: F, sol?: Boolean)) -> Union(LaurentPolynomial(F, UP), failed)
expintfldpoly(p, foo) returns q such that p' = q or “failed” if no such q exists. Argument foo is a Risch differential equation function on F.
explogint: (Fraction UP, UP -> UP, (Integer, F) -> Record(ans: F, right: F, sol?: Boolean), List Fraction UP) -> Union(Record(answer: Record(mainpart: Fraction UP, limitedlogs: List Record(coeff: Fraction UP, logand: Fraction UP)), a0: F), failed)
explogint(f, ', foo, [u1, ..., un]) returns [v, [c1, ..., cn], a] such that ci' = 0, f = v' + a + reduce(+, [ci * ui'/ui]), and a = 0 or a has no integral in F. Returns “failed” if no such v, ci, a exist. Argument foo is a Risch differential equation function on F.

primintegrate(f, ', foo) returns [g, a] such that f = g' + a, and a = 0 or a has no integral in UP. Argument foo is an extended integration function on F.
