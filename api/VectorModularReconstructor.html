This domain supports modular methods based on evaluation and rational reconstruction. All computation are done on polynomials modulo machine sized prime p -- p must be choosen small enough to avoid overflow in intermediate calculations. Each evaluation is supposed to produce vector of values. Once enough evaluations are known rational reconstruction produces vector of rational functions or multivariate polynomials.
