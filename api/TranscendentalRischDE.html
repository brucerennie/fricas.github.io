Risch differential equation, transcendental case. Author: Manuel Bronstein Date Created: Jan 1988 Date Last Updated: 2 November 1995

SPDEnocancel1: (UP, UP, Integer, UP -> UP) -> Record(ans: UP, nosol: Boolean)
SPDEnocancel1(b, c, n, D) solves Q' + b Q = c with deg(Q) <= n and returns [Q, flag] where flag is false when Q satisfies the equation, if flag is true Q is a partial solution. D is the derivation to use. Note: SPDEnocancel1 assumes that deg(Q') < deg(bQ) for all Q.
