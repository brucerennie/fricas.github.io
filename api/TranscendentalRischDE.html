Risch differential equation, transcendental case. Author: Manuel Bronstein Date Created: Jan 1988 Date Last Updated: 2 November 1995

baseRDE: (Fraction UP, Fraction UP) -> Record(ans: Fraction UP, nosol: Boolean)
baseRDE(f, g) returns a [y, b] such that y' + fy = g if b = false, y is a partial solution otherwise (no solution in that case). D is the derivation to use.
monomRDE: (Fraction UP, Fraction UP, UP -> UP) -> Union(Record(a: UP, b: Fraction UP, c: Fraction UP, t: UP), failed)
monomRDE(f, g, D) returns [A, B, C, T] such that y' + f y = g has a solution if and only if y = Q / T, where Q satisfies A Q' + B Q = C and has no normal pole. A and T are polynomials and B and C have no normal poles. D is the derivation to use.
polyRDE: (UP, UP, UP, Integer, UP -> UP) -> Union(ans: Record(ans: UP, nosol: Boolean), eq: Record(b: UP, c: UP, m: Integer, alpha: UP, beta: UP))
polyRDE(a, B, C, n, D) returns either: 1. [Q, b] such that degree(Q) <= n and a Q'+ B Q = C if b = false, Q is a partial solution otherwise. 2. [B1, C1, m, \alpha, \beta] such that any polynomial solution of degree at most n of A Q' + BQ = C must be of the form Q = \alpha H + \beta where degree(H) <= m and H satisfies H' + B1 H = C1. D is the derivation to use.
SPDEnocancel1: (UP, UP, Integer, UP -> UP) -> Record(ans: UP, nosol: Boolean)
SPDEnocancel1(b, c, n, D) solves Q' + b Q = c with deg(Q) <= n and returns [Q, flag] where flag is false when Q satisfies the equation, if flag is true Q is a partial solution. D is the derivation to use. Note: SPDEnocancel1 assumes that deg(Q') < deg(bQ) for all Q.
