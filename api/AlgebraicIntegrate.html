This package provides functions for integrating a function on an algebraic curve.

algintegrate: (R, UP -> UP) -> IntegrationResult R
algintegrate(f, d) integrates f with respect to the derivation d.
palgextintegrate: (R, UP -> UP, UPUP, Fraction UP, Fraction UP) -> Union(Record(ratpart: R, coeff: F), failed)
palgextintegrate(f, d, g, c1, c2) returns functions [h, c] such that using c1, c2 as a change of variables we have (d(h))(c1(x), c2(y)) = d(c1(x))f(c1(x), c2(y)) - c g(x, y), where y is an algebraic function of x; returns “failed” if no such functions exist.
palginfieldint: (R, UP -> UP) -> Union(R, failed)
palginfieldint(f, d) returns an algebraic function g such that dg = f if such a g exists, “failed” otherwise. Argument f must be a pure algebraic function.
palgintegrate: (R, UP -> UP) -> IntegrationResult R
palgintegrate(f, d) integrates f with respect to the derivation d. Argument f must be a pure algebraic function.
